Interface

Interface(name, methods)

A helper class to provide JavaScript with the foundational component of reusable Object Oriented Programming techniques.

Constructor

# new Interface(name, methods)

Parameters:
Name Type Description
name string

The name of the Interface
methods Array.<string>

A non-empty array of expected method names the given interface must implement

Example
const Composite = new Interface('Composite', ['add', 'remove', 'getChild']);
const FormItem = new Interface('FormItem', ['save']);

// CompositeForm class
const CompositeForm = function(id, method, action){
  // implements Composite, FormItem ...
